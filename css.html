Element selector:-

back

    The element selector chooses one or more HTML elements of the same name. Thus, a selector of nav chooses all HTML nav elements, and a selector of < ul > chooses all HTML unordered lists, or < ul > elements. In this ,selector chooses all elements with the specified element name.

    • Examples
    h2 {
    background-color:blue;
    }


    p {
    color:crimson;
    text-align:center;
    }

font-style-css:-

back

    Font Style. The font-style property is mostly used to specify italic text. This property has three values: normal - The text is shown normally , italic - The text is shown in italics; oblique - The text is "leaning" (oblique is very similar to italic, but less supported)

Background-image-css:-

back

    background-image The background-image CSS property sets one or more background images on an element. The background images are drawn on stacking context layers on top of each other. The first layer specified is drawn as if it is closest to the user.

    • Examples

    syntax = background-image:url("paste your image path");

    .bg
    {
    background-image: url('Ashoka_Chakra.png');
    background-size: cover;
    background-size: contain;
    background-size: 1000px 550px;
    background-repeat: no-repeat;
    background-position: right;
    }

Types of Background-gradient:-

back

    Gradient Background. CSS gradients let you display smooth transitions between two or more specified colors. CSS defines two types of gradients: Linear Gradients (goes down/up/left/right/diagonally) Radial Gradients (defined by their center)

    • Examples

    1 = Linear gradient
    2 = Radial gradient
    3 = Conic-gradient


    .linear-gradient
    {
    background:linear-gradient(to top right,red,orange,yellow,lightgreen,green)
    }

    .radial-gradient
    {
    background:radial-gradient(circle,white,black,white,red)
    }

    .radial-gradient
    {
    background:radial-gradient(ellipse,white,black,white,red)
    }

    .Conic-gradient
    {
    height: 200px;
    width: 200px;
    background-color: red; /* For browsers that do not support gradients */
    background-image: conic-gradient(red 0deg, blue, yellow 90deg,pink, green 180deg, gray, blue 270deg);
    border-radius: 50%;
    }
