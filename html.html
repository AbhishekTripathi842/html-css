Dropdown-list :-

back

    A drop-down list (abbreviated drop-down, or DDL;[1] also known as a drop-down menu, drop menu, pull-down list, picklist) is a graphical control element, similar to a list box, that allows the user to choose one value from a list. When a drop-down list is inactive, it displays a single value. When activated, it displays (drops down) a list of values, from which the user may select one. When the user selects a new value, the control reverts to its inactive state, displaying the selected value.

    • Examples
  • < select >
  • < option > anything < /option >
  • < option > anything < /option >
  • < /select >

Difference between checkbox and radio :-

back

    A checkbox (check box, tickbox, tick box) is a Graphical widget that permits the user to make a binary choice, i.e. a choice between one of two possible mutually exclusive options. For example, the user may have to answer 'yes' (checked) or 'no' (not checked) on a simple yes/no question.

    • Examples
  • < input type="checkbox" name = "chess" >chess
  • < input type="checkbox" name = "ludo" >ludo
  • < input type="checkbox" name = "match" >match
  • < input type="radio" name = "opt" >chess
  • < input type="radio" name = "opt" >ludo
  • < input type="radio" name = "opt" >match

HTML Tables with colspan and rowspan:-

back

    What does colspan= do? Allows a single table cell to span the width of more than one cell or column. What does rowspan= do? Allows a single table cell to span the height of more than one cell or row.

    • Examples
    < table >
    < thead colspan="3" >
    < tr >
    < th > anything < /th >
    < th > anything < /th >
    < th > anything < /th >
    < /tr >
    < /thead >
    < tbody >
    < tr >
    < td > anything < /td >
    < td colspan="2"> anything < /td >
    < td > anything < /td >
    < /tr >
    < tr >
    < td rowspan="2" > anything < /td >
    < td > anything < /td >
    < td > anything < /td >
    < /tr >
    < /tbody >
    < /table >
